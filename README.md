# Dados_INEP_PROFESSORES

## Projeto de Estrutura de Dados utilizando a biblioteca Pandas e Matplotlib diretamente no Jupyter Notebook

Com o intuito de realizar estruturação e análise de dados públicos, contamos com a linguagem Python e a biblioteca Pandas com auxílio da biblioteca Matplotlib para a realização de matrizes e plotagem de dados de forma gráfica, respectivamente.

Utilizando os dados públicos do INEP, disponibilizados pelo Governo do Federal, selecionamos os profissionais da educação da região sudeste com pelo menos uma especialização.

Com base no site do INEP, um banco de dados com todoas as informações referente ao censo escolar do Brasil, na sessão de "Microdados", selecionamos a área do censo escolar dos anos 2015 a 2018. http://portal.inep.gov.br/microdados Acesso em 5/12/2019.

O intuito desde trabalho foi demonstrar, com as informações obtidas por meio do site INEP, a quantidade de profissionais da região sudeste que, por faixa etaria, demonstram maior desenvolvimento acadêmico.
